# Calculatrice

Une calculatrice simple en ligne de commande écrite en JavaScript .

## Fonctionnalités

- Addition
- Soustraction
- Multiplication
- Division
- Calculs avec des nombres à virgule flottante
- Gestion des erreurs (division par zéro, entrées invalides)

## installation

- Clonez ce dépôt ou téléchargez les fichiers.

``git clone https://github.com/yassir-lazreq/calculatrice.git``

## structure des fichiers
- `calcul.html` : Le fichier HTML principal contenant la structure de la calculatrice.
- `calcul.js` : Le fichier JavaScript contenant la logique de la calculatrice.
- README.md : Ce fichier de documentation.
